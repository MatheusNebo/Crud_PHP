# SISTEMA CRUD

## 📋 Sobre
Sistema de controle de vendas em desenvolvimento para aprimorar conhecimentos em:

- CRUD (Create, Read, Update, Delete)  
- Consultas SQL com JOINs  
- Relacionamento entre tabelas
- Importação de arquivos CSV (em desenvolvimento)
- FrontEnd simples em HTML e CSS

## Funcionalidades já implementadas

- Banco de dados estruturado (tabelas: usuarios, produtos, clientes, vendas, venda_itens)
- Página de dashboard com relatório de vendas do mês
- Listagem básica de usuários
- Listagem básica de produtos
- Funções de exemplo de consulta prontas

## Próximos passos
- CRUD de clientes
- CRUD de vendas
- Relatório de vendas por período
- Importação e exportação de outras tabelas para CSV (atualmente possui apenas de produtos)
- Autenticação de usuários
- Nova versão refatorada utilizando POO, sanitizações e validações mais consistentes
